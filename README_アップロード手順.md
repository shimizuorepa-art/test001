# 最終アップ用フォルダ

このフォルダ内の2つが、提出先ごとの最終アップロード用です。

## タニコー用

アップするフォルダ:

`01_タニコー用_アップロード`

中の `index.html` がトップページです。

主な追加内容:

- 日々の業務管理システム サンプル
- 事務手順標準化サンプル
- 価格表・帳票確認サンプル
- 営業支援実績ビュー
- 受発注システム電子化サンプル
- 人員管理台帳サンプル

## 新東工業用

アップするフォルダ:

`02_新東工業用_アップロード`

中の `index.html` がトップページです。

主な追加内容:

- 販促・問い合わせ対応ダッシュボード
- 情報整理・回答下書きサンプル
- 商品情報・条件整理サンプル
- 営業支援・実績可視化サンプル
- 入力フォーム・管理画面サンプル
- 一覧・検索UIサンプル

## Vercelで使う場合

- Framework Preset: `Other`
- Build Command: 空欄
- Output Directory: 空欄
- タニコー用と新東工業用は別プロジェクトとして、それぞれ上記フォルダをルートにする

## 注意

- この親フォルダ全体ではなく、提出先に応じて `01_...` または `02_...` の中身をアップする。
- どちらもサンプル用のダミーデータを使用し、機能や表示は簡略化済み。
- Git commit / push はこの作業では行っていない。
