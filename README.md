# AI・Web販促活用サンプルサイト

新東工業のマーケティング職応募に向けた、AI・Web販促活用の補足サンプルサイトです。

実務で作成した画面や資料をもとに、情報整理、検索、見える化、下書き作成の考え方を簡素化してまとめています。特定の運用業務をそのまま行いたいという趣旨ではなく、Webコンテンツ改善、カタログサイト整理、営業資料の整備、展示会後フォローなどへ転用できる可能性を示すための補足資料です。

## 構成

- `index.html`: 統合トップページ
- `demos/admin_standardization/`: 情報整理・回答下書きサンプル
- `demos/price_check/`: 商品情報・条件整理サンプル
- `demos/sales_support_dashboard/`: 営業支援・実績可視化サンプル
- `demos/marketing_activity_dashboard/`: 販促・問い合わせ対応ダッシュボード
- `demos/order_system/`: 入力フォーム・管理画面サンプル
- `demos/attendance_checkin/`: 出席事前登録・当日受付アプリ
- `demos/booking_reservation/`: 予約フロー画面バリエーション
- `demos/foreign_staff_database/`: 一覧・検索UIサンプル

## Vercel公開

このフォルダをGitHubへアップし、Vercelでプロジェクト化すれば、ルートURLの `index.html` から全サンプルを確認できます。

ビルドコマンドは不要です。Framework PresetはOther、Output Directoryは空欄のままで問題ありません。

## 注意

各サンプルは、実務の中で制作した画面や資料をもとに、応募補足用として加工・簡素化しています。社名、組織名、氏名、数値などはサンプルデータへ差し替えています。実顧客情報、現職の機密情報は含めない前提です。
