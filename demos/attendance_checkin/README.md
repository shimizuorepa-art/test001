# 出席登録・当日受付アプリ

出席事前登録、開発者メンテ、当日受付を確認するためのサンプルです。

## 画面

- `index.html`: 受付入力、開発者画面、当日画面をタブで切り替える統合デモ

## サンプル化方針

- 団体名、来場者名、席順、オーダー内容はサンプル内容へ置換しています。
- 受付、開発者、当日画面の流れを1画面で確認できます。

## 運用説明

- 受付入力画面: 事前に出席者の名称をセットし、席順や配置も視認できます。
- 開発者画面: キャンセル対応やボタンのブロックなど、後々発生するメンテナンス事項に対応します。
- 当日画面: 来場者がアプリ一つで出席登録から会場のオーダーまでできる想定です。
