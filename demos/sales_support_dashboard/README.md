# 営業支援実績ビュー

営業担当者からの「取引先ごとに実績を確認したい」という要望に応えて作成した画面を、応募補足用に匿名化・サンプルデータ化した静的サンプルです。

## 元の経験として伝えること

- 営業担当者の要望を聞き取り、必要な指標と比較軸を整理した
- 取引先別、月別、前年差、目標比を一画面で確認できる画面案を作成した
- 画面を見ながら、営業担当の次アクションや改善案を検討できる状態にした
- 実データや社名・場所名は使わず、応募補足用には完全なサンプルデータへ置き換えている

## マーケティング職向けの接続

このサンプルは、営業・販促活動で使う情報を整理し、次のアクションを見つけやすくする考え方を示す補足資料です。

- 取引先別の実績可視化
- 伸長・減少の早期把握
- 営業担当への次アクション案の整理
- 商談前準備や展示会後フォローに使える1枚メモ化
- 社内の要望を確認軸へ分解し、小さく試作して運用検討へ進める力

## 開き方

`index.html` をブラウザで開くと動きます。外部通信はありません。
