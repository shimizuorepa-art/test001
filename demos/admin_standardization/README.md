# 情報整理・回答下書きサンプル

Web販促・営業支援で使う情報整理に合わせた静的サンプルです。問い合わせ内容、商品情報、社内資料、過去FAQなどを整理し、回答案や確認事項の下書きへ落とし込む考え方を示します。

このサンプルは、特定の実データを再現したものではありません。実際のマーケティング業務や社内ルールに応じて、入力欄、確認項目、出力文、表示項目をカスタムできる前提です。

## 入っているツール

- 問い合わせ内容の整理
- 確認項目チェック
- FAQドラフトメーカー
- 社内資料参照・回答下書き
- 期限・進捗ボード

各ツールを選ぶと、画面上部に「想定される使用場面」と「この画面でできること」が表示されます。
下段の入力画面で実際に操作すると、完了内容が黒い通知ラベルで表示されます。

## 設計意図

課題: 製品情報、社内資料、過去の問い合わせ、営業資料が分散していると、Web掲載や販促資料へ落とし込むまでに時間がかかりやすい。

仕組み: 入力内容を、対象者、確認項目、根拠資料、回答文、次アクションに分解して表示します。

期待効果: Webコンテンツや販促資料へ転用しやすい形で、情報を整理できます。

自分の工夫: 商品情報整理で得た「正本を決め、参照ルールを固定し、見る人が使いやすい形に戻す」経験を、販促・営業支援にも転用できる形にしています。

## 社内資料参照の説明

専門用語は前面に出さず、「社内資料、最新版、確認者、例外条件を整理し、根拠つきで回答や掲載案の下書きを作る仕組み」と説明する想定です。

サンプル内では、問い合わせ回答、確認事項整理、FAQ化などのパターンを用意しています。出力は確定処理ではなく下書きで、社外発信に関わる判断は必ず人が確認する前提です。

## 開き方

`index.html` をブラウザで開くと動きます。外部通信はありません。入力内容や参照データはすべてサンプルです。

## 応募での見せ方

AIや簡易ツールを使って情報を整理し、Web施策や販促資料へ転用できる準備ができることを示す成果物です。
